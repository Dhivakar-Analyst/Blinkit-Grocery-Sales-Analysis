# 🛒 Blinkit Sales Dashboard | Microsoft Excel

## 📌 Project Overview

This project is an interactive **Blinkit Sales Dashboard** built in **Microsoft Excel** to analyze sales performance, customer preferences, outlet performance, and product categories.

The dashboard provides business insights using Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and Excel Dashboard Design techniques. It enables users to explore sales trends and make data-driven business decisions.

---

## 🎯 Business Objective

The objective of this project is to analyze Blinkit's sales data and answer key business questions such as:

- Which outlet size generates the highest sales?
- Which outlet location performs the best?
- Which item categories contribute the most revenue?
- How do sales vary by outlet establishment year?
- What is the average sales value and customer rating?
- Which outlet type has the highest total sales?

## 📊 Business Insights

### 🟢 Outlet Performance

- Medium-sized outlets generated the highest sales.
- Tier 3 locations recorded the highest revenue.
- Supermarket Type 1 contributed the largest share of sales.

### 🟡 Product Performance

- Fruits & Vegetables generated the highest sales.
- Snack Foods ranked as the second-highest category.
- Seafood recorded the lowest sales.

### 🔵 Sales Trend

- Sales peaked around the 2018 outlet establishment period.
- Sales remained relatively stable across most establishment years.

### 🟣 Customer Insights

- Average customer rating is **3.9**.
- Average sales per transaction are **$141**.
